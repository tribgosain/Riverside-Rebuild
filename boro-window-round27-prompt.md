# Riverside Rebuild — Round 27: Premium Editorial Polish Pass

Paste into Claude Code. This keeps the newspaper/tabloid identity —
that's confirmed as the right direction — but refines it toward "premium
editorial" (think The Athletic, the FT, a well-designed magazine site)
rather than "busy tabloid front page." The core principle: premium
editorial design uses MORE restraint and whitespace than a cheap tabloid
look, not more decoration. A signature detail used once with confidence
reads as expensive; the same detail repeated everywhere reads as trying
too hard.

## 1. Significantly more whitespace

Increase spacing throughout — take the spacing scale established in
round26 and scale it up, roughly 1.5x across the board. More breathing
room around every section, more generous margins, more space between the
hero photo and the content below it, more space around the name input
and CTA. This is the single highest-leverage change here.

## 2. Reconsider the halftone dot texture specifically

A visible dot-matrix pattern over the photo can read as a heavy-handed
Photoshop filter rather than genuine refined photo treatment. Try
significantly reducing its opacity, or removing it and keeping just the
clean duotone (ink/cream) treatment alone — duotone by itself is often
more premium-reading than duotone-plus-visible-halftone-dots. Compare
both and use whichever actually looks more considered, not automatically
whichever is closer to the original spec.

## 3. Consolidate decorative elements — use fewer, more deliberately

Audit how many hairline rules, dashed lines, and small decorative marks
appear on this one screen. If the girder motif or divider rules appear
more than once, consolidate to a single, confident use rather than
scattering the same device across multiple spots. Fewer decorative
touches, each doing real work, reads as more considered than many small
ones.

## 4. Tighten the display headline's letter-spacing

Large display type (the "RIVERSIDE REBUILD." headline at its current
size) typically needs slightly negative letter-spacing/tracking to read
as refined rather than loose — this is a real typographic principle, not
a stylistic guess: the bigger the type, the tighter the tracking needs to
be relative to body text. Check and tighten this specifically.

## 5. Use the red accent more sparingly

Audit every place red currently appears on this screen. Red should read
as a deliberate signal — reserve it for a small number of genuinely
important moments (the "REBUILD" word, the CTA, maybe one accent rule)
rather than appearing on multiple secondary elements at once. If red is
currently doing decorative work in more than 2-3 places, pull it back.

## 6. Check strict alignment throughout

Every text block, label, and element on this screen should align to one
consistent left edge / grid, not slightly varying indents between
sections. This is a small thing individually but adds up to a real sense
of precision when it's consistent, and a real sense of sloppiness when
it's not.

## Not required this round, mention if there's appetite

A subtle, tasteful entrance animation (content gently fading/settling in
on load) is a common premium-site touch — but it's a bigger technical
lift with more room to go wrong given this session's track record with
ambitious asks. Only attempt this after everything above is done and
confirmed looking right; don't let it distract from the more
foundational fixes above.

## Definition of done

- Spacing visibly more generous throughout, not just technically
  "increased" by a small amount
- Halftone dot treatment reconsidered — compare with/without, keep
  whichever reads as more premium
- Decorative elements (rules, girder motif) consolidated to fewer, more
  deliberate uses
- Headline tracking tightened
- Red accent audited and pulled back to a small number of deliberate uses
- Alignment checked and made consistent throughout
- Screenshot the result at both desktop and mobile before calling it done
